# 错误处理

大多数错误并不严重到需要程序完全停止。
有时，当一个函数失败时，其原因可以被你轻松地解释和响应。
例如，如果你尝试打开一个文件而该操作因文件不存在而失败，你可能希望创建该文件而不是终止进程。
